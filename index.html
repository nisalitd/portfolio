VEXEL: An Autonomous Mobile Robotic Manipulator for Smart Agriculture

VEXEL is an autonomous mobile manipulator developed for disease detection and targeted spraying in confined polytunnel environments. It integrates on-board crop monitoring with adaptive spraying, enabling plant-level treatment while navigating autonomously through narrow rows. The system unifies navigation, perception, and manipulation in a single platform, allowing autonomous detection-to-actuation without human intervention.

Biomimetic Underwater Robot for Water-Quality Monitoring in Aquaculture

A cuttlefish-inspired underwater robot designed for autonomous water-quality monitoring in aquaculture environments. The system employs a biomimetic undulatory fin mechanism driven by dual cam-shaft actuators, enabling smooth and efficient MPF-based propulsion. The locomotion mechanism was developed through functional 3D design and validated via 2D hydrodynamic simulations using dynamic meshing and user-defined functions. Motor, battery, and control components were selected for compact, energy-efficient operation in shallow-water conditions.

Line/Wall Following Robot with Embedded Road Sign Recognition

A differential-drive mobile robot that follows a black path using an 8-IR PID controller, avoids obstacles via ultrasonic sensing, and executes junction turns based on hybrid vision recognition. A HOG–Random Forest classifier detects symbolic signs (BUS STOP, P, TRAIN) with OCR fallback for numeric speed signs (20, 100), enabling adaptive speed control and junction decisions. All processing runs on a Raspberry Pi with a Flask-based web and mobile interface supporting manual, line-follow, and obstacle-assist modes for seamless control and monitoring.

Vision-Guided Cartesian Crop Harvesting Robot

A vision-assisted Cartesian robot developed as a proof-of-concept for automated crop harvesting. The system identifies ripe and unripe crops by color, locates stem positions in a fixed coordinate frame, and mimics a harvesting motion using a servo-actuated gripper mounted on a stepper-driven Cartesian stage. Image processing and object detection were implemented using Python/OpenCV and YOLOv5 (Roboflow-trained), with SQLite database integration and Raspberry Pi–Arduino control for coordinated motion execution.

Modular Automated Production Line

A PLC-controlled smart factory line featuring conveyors, changer modules, stack magazines, and dual stamping stations with adjustable stamp counts and timing for each part type. Color-based detection using fiber-optic sensors enables automatic routing and station selection, while a centralized HMI provides real-time visualization, parameter control, and alarm monitoring. Faults are identified through post-process height analysis, automatically flagging and notifying defective parts.
